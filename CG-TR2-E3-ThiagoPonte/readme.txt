Para executar, abrir o arquivo TR2_E3_ThiagoPonte.html no Chrome ou Firefox.

O código está dividido em alguns arquivos.

app.js 			-- O principal, contém funções para desenhar os objetos.
				-- Contém o código de atualização da scene.
				-- Para abrir e fechar os dedos, utilizar os controles no canto superior direito.

Os outros arquivos contêm código auxiliar:
fpsStats.js 	-- contador de FPS.
setup.js 		-- Inicializa o renderer, camera e scene.

Arquivos dentro da pasta "js", são do WebGL.
